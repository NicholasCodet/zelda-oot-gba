# Zelda Ocarina of Time – GBA Adaptation

## Overview

This project aims to create a **2D Game Boy Advance adaptation inspired by Ocarina of Time**.

The goal is not to recreate the Nintendo 64 game exactly, but to reinterpret its world, mechanics and progression in a **top-down 2D format compatible with the Game Boy Advance hardware**.

The gameplay approach is inspired by classic 2D Zelda games such as:

- The Legend of Zelda: A Link to the Past
- The Legend of Zelda: The Minish Cap

This project explores what **Ocarina of Time might have looked like if it had been designed directly for the Game Boy Advance**.

---

## Current Status

The project is currently in the initial setup phase.

Goal:
- create a minimal GBA project
- compile a ROM
- run it in an emulator

## Project Goals

The project focuses on three main goals.

### Adaptation

Reinterpret the locations, story structure and mechanics of Ocarina of Time in a 2D tile-based environment.

### Technical Exploration

Build a small **Zelda-like engine for the Game Boy Advance**.

### Incremental Development

Develop the game through **small vertical slices** rather than building the entire game at once.

## Development Roadmap

### Phase 1 – Engine foundation

- display a tilemap
- player movement
- tile collisions

### Phase 2 – World interaction

- dialogue box
- triggers
- doors
- simple items

### Phase 3 – Combat

- sword attack
- enemies
- damage system
- health system

### Phase 4 – First playable area

A small overworld inspired by **Kokiri Forest**.

Includes:

- exploration
- NPC interactions
- interior buildings
- basic combat

### Phase 5 – Dungeon prototype

A small dungeon used to validate puzzle and progression systems.

Includes:

- keys
- locked doors
- switches
- mini boss
